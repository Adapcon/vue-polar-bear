# **PbSkeletonLoading Component**

### Description

The PbSkeletonLoading component is a component that shows a skeleton loading.

### Properties

| Property                      | Type                 | Default       | Description
|:------------------------------|:---------------------|:--------------|:-------------------------------
| mainContainerDimensions       | Object               | {}            | Dimensions of the main container. This property can receive values just like css attributes would receive (px, rem, em ch, %). Example: { width: '320px', height: '180px' }
| secondaryContainerDimensions  | Object               | {}            | Dimensions of the secondary container. This property can receive values just like css attributes would receive (px, rem, em ch, %). Example: { width: '80px', height: '30px' }
| secondaryDirection            | String               | 'row'         | Direction of the secondary container, 'row', 'column', 'row-reverse', 'column-reverse'
| containersAlign               | String               | 'flex-start'  | Alignment of the containers, 'flex-start', 'flex-end', 'center', 'space-between', 'space-around', 'space-evenly'
| mainContainerBorderRadius     | Number               | 0             | Radius of the main container
| marginBetweenItems            | Number               | 0             | Margin between the main and secondary container
