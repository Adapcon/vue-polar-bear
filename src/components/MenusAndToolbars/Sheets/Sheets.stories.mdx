# **PbSheets Component**
### The `PbSheets` component is used to display multiple buttons in just one.

### Purpose and Use case

has the proposal to display several buttons in just one. component has an overlay that overlays the entire screen. you can pass an object containing the name of the option and within that option the title of the button and its icon
