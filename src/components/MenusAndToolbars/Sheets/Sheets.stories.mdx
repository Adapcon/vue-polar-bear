# **PbSheets**
### The `PbSheets` is a surface containing supplementary content that are anchored to the bottom of the screen.

### Purpose and Use Case

Probably you'll want to use PbSheets on mobile navigation.

## Configuration

### Properties

| Property                | Type      | Default            | Description
|:------------------------|:----------|:-------------------|:--------------------------------------------------------------------------------------|
| sheets                  | Array     | Required           | An array of content objects, it includes the label, icon and value. Icon is unrequired|
| buttonIcon              | String    | fas fa-ellipsis-v  | Button icon                                                                           |
| buttonColor             | String    | primary            | Button color                                                                          |
| buttonStyle             | String    | regular            | Button style                                                                          |

### Events

| Event                       | Description
|:----------------------------|:-----------------------------------------|
| selectSheet                 | Fired right after selection of a sheet   |
