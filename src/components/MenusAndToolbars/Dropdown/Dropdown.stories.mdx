# **PbDropdown**
### The `PbDropdown` is a simple dropdown component capable of rendering content inside a floating box.

### Purpose and Use Case

This component can be used to show more options inside a button or any kind of element you wish.
It accepts two slots, one named *header* for any kind of element that will be render at the first time
and it will have the action to activate the dropdown, and the *default* one that is all the dropdown content.

If a *header* slot is not passed, the component will try to render the text comming from the prop.

## Examples

          <PbButton button-style="background" color="primary" label="Sign out" />
        </div>
      </PbDropdown>
    </div>
  `,
