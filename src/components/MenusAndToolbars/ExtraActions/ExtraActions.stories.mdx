# **Pb-Extra-Actions Component**
PbExtraActions is a component that renders extra action buttons on the screen.

#### **Purpose and Use case**
This component was developed to make it possible to render extra action
buttons in other components in order to make interfaces cleaner.
It works similarly to a toolbar, with a initial state defined by the prop `initialState` (with 'closed' as default).
When triggered it animates in a left-to-right direction all the content passed through the slot.

## `Animation Disclaimer`
To make animations work properly, every Vue component should be wrapped by a regular Html tag.
Also, if you face issues like the content taking too long to render, try to unable animations.
This might occur when long processes are added to the javascript task queue.
