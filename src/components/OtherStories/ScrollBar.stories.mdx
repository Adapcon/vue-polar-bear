# **PbScrollBar**
#
If you need a narrower scroll bar to your element or page you can use three CSS classes specifically for this purpose.


### **Primary Color ScrollBar**
Use the `pb-scroll-primary` class on your element to achieve this result.

<Canvas>
  <div
    class="pb-scroll-primary"
    Style="width: 100%; height: 200px; overflow: auto;"
  >
    <div Style="height: 400px">
      <p class="pb">
        The scroll bar is on the right side of this box, check it out by scrolling me...
      </p>
    </div>
  </div>
</Canvas>

---
