# **PbGlobalClasses**
#

Several other global classes and animations have been added to the `animations.scss` and `index.scss` files, here's a list of them:

### **Global Classes**
#

**`cursor-disabled`** Changes the cursor to **not-allowed**.

**`disable-select`** Disable text selection.

**`bg-color-primary`** Switch the background color to **primary**.

**`bg-color-secondary`** Switch the background color to **secondary**.

**`bg-color-success`** Switch the background color to **success**.

**`bg-color-danger`** Switch the background color to **danger**.

**`bg-color-warning`** Switch the background color to **warning**.

**`bg-color-info`** Switch the background color to **info**.


---

### **Global Animations**
#

**`border-spin`** A border that spin.

**`shake`** The element shakes horizontally.

**`light-gray-blink`** A light gray background that blinks.

**`opacity-blink`** Blinks the opacity from 0.3 to 1.

**`pulse-loading`** Creates a pulsing effect for PbPulseLoading.
