# **PbColors**
#

To use the **PbColors** you should call the scss variables by using `var(--color-colorName)`
The only colors allowed to be used on the Pb UI are the ones following.
If new colors are necessary they should be added to the `app.scss` file.


### **Client Colors**

#### **`var(--color-primary)`**
This variable applies the **client primary color** to the element.

#### **`var(--color-secondary)`**
This variable applies the **client secondary color** to the element.

#### **`var(--color-active)`**
This variable applies the **client active color** to the element.

---

### **Background Colors**

#### **`var(--color-background-dark)`**
This variable applies the **#484F59** to the background.

---

### **App Colors**

#### **`var(--color-success)`**
This variable applies the **#417617** to the element.

#### **`var(--color-danger)`**
This variable applies the **#ac4545** to the element.

#### **`var(--color-warning)`**
This variable applies the **#dd8218** to the element.

#### **`var(--color-info)`**
This variable applies the **#18a0fb** to the element.

---

### **Gray Scale Colors**

#### **`var(--color-white)`**
This variable applies the **#fff** to the element.

#### **`var(--color-gray)`**
This variable applies the **#f8f8f8** to the element.

#### **`var(--color-gray-10)`**
This variable applies the **#ccc** to the element.

#### **`var(--color-gray-20)`**
This variable applies the **#aaa** to the element.

#### **`var(--color-gray-40)`**
This variable applies the **#888** to the element.

#### **`var(--color-gray-60)`**
This variable applies the **#555** to the element.

#### **`var(--color-gray-80)`**
This variable applies the **#333** to the element.

#### **`var(--color-gray-90)`**
This variable applies the **#222** to the element.
