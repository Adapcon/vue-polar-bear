# **PbToggleSwitch**
### The `PbToggleSwitch` component can be used to select or Toggle one option.

### Purpose and Use case

It is a simple Toggle Switch field that can be used to provide a way to select one option like confirm terms and conditions.

The component works with v-model to emit the checked value.

## Configuration

### Properties

| Property                | Type                 | Default | Description
|:------------------------|:---------------------|:--------|:-------------------------------
| selection               | Boolean              | false   | default value of the Toggle

### Events

| Event                       | Description
|:----------------------------|:---------------------------------------------------------------------
| updateSelection             | change value of toggle.
