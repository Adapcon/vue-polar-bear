# **PbDocumentInput**
### The `PbDocumentInput` component is used as input for get official company register document.

### Purpose and Use Case
Its main purpose is to group input and validation together with a component.

### Usage
With `inputType`, you can select the type of document you want to use as input, and the placeholder will be automatically set, unless you want to use a custom placeholder.
You can also set two different input types, to support more documents.

When the `inputType`, has more than one type, the placeholder will be empty, unless you set a custom placeholder.

If you want to use a custom placeholder, you can set it using the `placeholder` prop.

If the input is required, you can set it using the `required` prop.

The `background` and `color` props are used to set the background and color of the input.
