# **PbDateInput component**
### The `PbDateInput` was created to capture date through user input.

### Purpose and Use case

The `PbDateInput` component is used to capture date input from the user.

The component has an `on-validate` property similar to the vue properties validator.
It is used to perform the validations of the date.
