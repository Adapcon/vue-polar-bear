# **PbSelect**
#

### The `PbSelect` component can be used as an option selector.

It was created to make it possible to select one or more options in a form context.
All the logic should be created from outside if the items should disappear when selected (in case of multiple selects)
or if the label should change (in case of single selection), for example.

When using it for multiple selections, try to combine with PbChips to display the selected options.
The `allowSearch` prop gives to the user the possibility of search between the options.
