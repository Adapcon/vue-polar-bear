# **PbDateRangePicker component**
### The `PbDateRangePicker` was created to capture a date range through user input.

### Purpose and Use case

The `PbDateRangePicker` component is used to capture a date range input from the user.

This component has some validations that don't allow the user to select an invalid date range.
For example, if the user selects a start date that is later than the end date, the component will not allow the user to select the end date and vice versa.

### Usage

```html
<PbDateRangePicker
  :value="{ startDate: '', endDate: '' }"
  :calendars-position="right | left"
  :input-style="background-light | outline"
/>
```

## Props

This component have the following props:

| Prop | Type | Default      | Description |
| ---  | ---  | ------------ | --- |
| value | Object | { startDate: '', endDate: '' } | Object used to set the value of the input, always have to pass the startDate and endDate as the usage example |
| calendars-position | String | right | Used to set the side that the calendary opens |
| input-style | String | background-light | Used to set the style of the input |
