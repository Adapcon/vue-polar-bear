# **PbCheckboxTree**
### The `PbCheckboxTree` component can be used to select or check options.

### Purpose and Use case

It is a tree of checkbox fields that can be used to provide a way to select one or more option like confirm terms and conditions

The component works with v-model to emit the checked value.
