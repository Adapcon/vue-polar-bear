# **PbAudioInput component**
### The `PbAudioInput` was created to capture texts through user input.

### Purpose and Use case

The `PbAudioInput` component is used to capture text input from the user.
It uses v-model, similar to a regular input.

The component has a `validator` property similar to the vue properties validator.
This property must contain all validations, if it is valid, return `true`
and if validation fails, it should return `false`.
This will cause all changes to the `value` to be validated and if an error
occurs visually modify the component.
