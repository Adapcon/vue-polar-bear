# **PbRadio**
### The `PbRadio` component can be used to show a list of radio inputs.

### Purpose and Use case

It's a simple Radio input list that can be used to provide a way to select one option in a option list.

The component works with v-model to emit the checked value.

## Configuration

### Properties

| Property                | Type                 | Default | Description
|:------------------------|:---------------------|:--------|:-------------------------------
| options                 | Array                | []      | Options list
