# **PbQuantitySelector**
### The `PbQuantitySelector` component can be used to manipulate quantities.

### Purpose and Use case

This component can be used when it's necessary to manipulate quantities in a simply and quickly way.
It has two buttons, one to add and another to subtract, making the user experience simple and easy.


This component returns feedback when errors occur.
The errors are based on `min` and `max` property, and in case the user inputs a value that's not a number.

There are two main styles for this component that can be attributed through the `quantitySelectorStyle` prop: `regular` and `outline`.
