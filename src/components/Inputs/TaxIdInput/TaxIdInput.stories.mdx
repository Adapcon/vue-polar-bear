# **PbTaxIdInput**
### The `PbTaxIdInput` component is used as input for get official company register document.

### Purpose and Use Case
Its main purpose is to group input and validation together with a component.

### Usage
With `allowedDocuments`, you can select the type of document you want to use as input, and the placeholder will be automatically set, unless you want to use a custom placeholder.
You can also set two different input types, to support more documents.

When the `allowedDocuments`, has more than one type, the placeholder will be empty, unless you set a custom placeholder.

If you want to use a custom placeholder, you can set it using the `placeholder` prop.

If the input is required, you can set it using the `required` prop.

The `background` and `color` props are used to set the background and color of the input.

### Properties

| Property                | Type                 | Default              | Description
|:------------------------|:---------------------|:---------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------|
| allowedDocuments        | Array                | ['cnpj']             | Array containing the types of documents. Never use a empty array.                                                                                   |            |
| required                | Boolean              | false                | Disable or enable the field requirement.                                                                                                            |
| background              | String               | transparent          | Set the background color of the input.                                                                                               |
| color                   | String               | primary              | Set the color of the input.                                                                                                                          |
| placeholder             | String               | 'Insira o número do documento!' | Set a custom placeholder.                                                                                                                            |
