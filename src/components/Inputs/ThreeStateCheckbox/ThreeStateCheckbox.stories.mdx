# **PbThreeStateCheckbox**
### The `PbThreeStateCheckbox` component can be used to select or check one option.

### Purpose and Use case

It is a simple checkbox field, with a third state, that can be used to provide a way to select one or more option like confirm terms and conditions,
or inform that has a list of options selected.

It has a label prop that can be used to pass a text to it.
The component works with v-model to emit the checked value.
