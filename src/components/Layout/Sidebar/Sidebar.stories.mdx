# **PbSidebar component**
### The `PbSidebar` component display the website's navigation.

### Purpose and Use case

The component's main purpose is to show the website's navigation.

It's fixed on the left side of the page and can be used to store filters and other essencial information as well.

The Sidebar have 3 main slots:
- Header extra action slot: can be used to show an extra action, like a button or dropdown
- Tabs slot: tabs are used to navigate between pages and this part will be fixed when the screen is mobile
- Content slot: here you can store everything the sidebar needs, such as filters and banners to display information between pages.

The Sidebar also provides a collapse system. If you enable it, the component can be closed so the main content of the page can be wider (this is extremely useful when dealing with tables with a lot of columns).

When mobile, the content of the Sidebar is hidden from the page until the user toggles it through the button besides the page title.
