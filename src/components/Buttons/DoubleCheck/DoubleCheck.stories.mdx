# **PbDoubleCheck**
### The `PbDoubleCheck` component can be used to confirm actions

### Purpose and Use Case

The double check component can be used in several places to be used as a confirmation of a certain action.
When the first click is made, the component will render a div containing the `confirmText`.
The confirmation div rendered by the component must overlap content on the left.
