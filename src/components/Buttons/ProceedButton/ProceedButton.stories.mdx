# **PbProceedButton**
### The `PbProceedButton` component can be used to give the idea of ​​continuity.

### Purpose and Use Case

The `PbProceedButton` component displays a button to continue the flow.
The concept of the component is to serve as a simple choice to continue some flow.
The button contains only the title and a fixed icon.
