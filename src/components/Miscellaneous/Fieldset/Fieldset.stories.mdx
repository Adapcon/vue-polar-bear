# **PbFieldset component**
### The `PbFieldset` component is a grouper of elements.

### Purpose and Use case

Its main purpose is to group one or more elements together with a label.
The PbFieldset component can be useful in several contexts, such as adding a label to an input,
It is also possible to use to group certain elements, as in a form for example.

## Examples

### Default

The component works in a simple way, having a prop and a slot.
All the elements that must be grouped can be passed through the slot.
