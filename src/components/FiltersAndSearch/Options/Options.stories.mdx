# **PbOptions component**
### The `PbOptions` component can be used as a simple selector of options.

### Purpose and Use case

The options component was designed to provide an input of choice in a simple way.
It is worth noting that the default for this component is that the selected option appears
with the **primary** color of the application.

## Code

It is an output component, that's why it's necessary to introduce a `v-model`.
This `v-model` will always return the **selected option**.

In order to use it's necessary to pass an array of string.

```
[
  'OPTION',
  'OPTION2',
  'OPTION3'
]
```
