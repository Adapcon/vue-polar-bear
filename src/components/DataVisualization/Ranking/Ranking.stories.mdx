# **PbRanking component**

### the `PbRanking` can be used as a simple list for displaying ranked informations.


### Use cases

You can use this component to display informations as a rank, in order. Most of the time for statistics and measurable values.

### Example:

<Canvas>
  <Story name="Default" height="auto">
    {Template.bind({})}
  </Story>
</Canvas>

### Code
The following structure is necessary in order to show the informations:
```
[
  {
    title: 'Some title',
    description: 'Some description' (optional),
    value: 'Some value' (optional)
  }
]
```

> **OBS:** This component does not has the responsibility of ordering the contents that is being pass as arguments
