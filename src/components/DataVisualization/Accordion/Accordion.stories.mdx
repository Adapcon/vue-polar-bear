# **PbAccordion component**
### The `PbAccordion` component was created to render an collapse for the user.

### Purpose and Use case

The component have a header with a collapse trigger, and inside it you could show
what you want inside the slot field.

You can select the quantity counter option to show the length of the element that you have
inside you Accordion too.

## Examples

### Default

export const Template = (args, { argTypes }) => ({
  props: Object.keys(argTypes),
  components: { PbAccordion },
  template: `
  <div>
    <PbAccordion
      :title="title"
    >
      <h4 class="pb" style="color: var(--color-gray-40); padding: 8px 14px;">Here you could code what you want!</h4>
    </PbAccordion>
  </div>
  `,
});
