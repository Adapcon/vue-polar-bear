# **PbChips**
### The `PbChips` component can be used to display selected options as chips.

### Purpose and Use case

This component was created to show a group of selected options, making it possible to remove unnecessary options one by one or all at once.

You can choose color and style for the chips and if you want to show the button/chip that removes all chips at once.
To use it you can just pass an array of strings at the `chips` property.

Remember to add the `.sync` modifier to the chip property, so the list options will auto-update every time user removes an item.
