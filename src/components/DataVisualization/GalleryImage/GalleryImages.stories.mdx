# **PbGalleryImage**
### The `PbGalleryImage` component can be used to display an image gallery.

### Purpose and Use Case

This component was created to display a preview gallery, where images can be easily seen.

## Code

You can pass an array of images via `:images=""`, pass sizes with the parameters, `:width=""`, `:height=""`, `:size=""`, colors to the icon with `:color=""`,
and the status of the transition, if it is true or false, `@toggleImages=""`

```
:images=""
:width=""
:height=""
:size=""
:color=""
@toggleImages=""

```
