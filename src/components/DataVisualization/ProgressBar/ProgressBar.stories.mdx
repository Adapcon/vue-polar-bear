# **PbProgressBar**
### The `PbProgressBar` component can be used to display a progress bar.

### Purpose and Use Case

The `PbProgressBar` component displays a colored progress bar.
The bar can contain labels to help the user know which step he is on.
This component should be used to inform the user of their progress in certain actions.
