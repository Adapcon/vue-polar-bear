# **PbToastNotification**
### The `PbToastNotification` component can be used to display quick and easy notification messages.

### Purpose and Use Case
The `PbTestNotification` component displays an animated toast with a simple message.
You can control through props the duration of the toast on screen, the message shown and the colors.
To show the component just call its `toggle()` method.
